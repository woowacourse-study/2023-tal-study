# ✨ tal-study
우아한테크코스 백엔드 5기 크루들의 가늘고 길게 가는 네트워크 스터디

<br>

## 🗓️ 기간 및 시간
* 2023년 4월 24일 ~
* 월요일 오전 11:00

<br>

## ✏️ 진행 방식
**1. 스터디 전**
* 매주 스터디전까지 강의 내용을 마크다운으로 정리하여 `PR` 을 보낸다.
* 스터디 하루 전날까지 각자 더 깊게 공부하고 싶은 부분이나 추가로 공부한 내용은 `Discussions`에 올린다.  

**2. 스터디**
* 강의 전체 내용 간단하게 정리 & 질문
* `Discussions` 내용 공유 및 토론 내용 기록
